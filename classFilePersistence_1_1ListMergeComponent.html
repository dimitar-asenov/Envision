Tries to find a unique position for elem in chunk that is similar to the position of elem in origin.

Returns a Position pos where pos.valid = true if and only if such a position could be found and pos.predecessor is the element after which elem should be inserted or 0 if elem should be inserted at the beginning. Such a position can be found if the nearest predecessor and successor of in origin that are common in chunk are next to each other and in order in chunk.
